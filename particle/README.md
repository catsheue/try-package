# particle

> Made with create-react-library

## Install

```bash
npm install --save particle
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'particle'
import 'particle/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
